# OptiCV.ai

OptiCV.ai is an AI-powered tool designed to help users generate optimized CVs for job applications, internships, and other contracts. The platform offers a user-friendly interface and a range of features to enhance the job application process.

## Features

- **AI-Driven CV Generation**: Utilize advanced AI algorithms to create tailored CVs that stand out to employers.
- **User Management**: Secure user authentication and profile management.
- **Subscription Plans**: Flexible subscription options to access premium features.
- **Payment Processing**: Seamless integration with payment gateways for easy transactions.
- **Responsive Frontend**: A clean and intuitive user interface built with modern web technologies.

## Project Structure

The project is divided into two main parts: the backend and the frontend.

### Backend

- **src**: Contains the main application code.
  - **controllers**: Handles the business logic for various functionalities.
  - **routes**: Defines the API endpoints.
  - **models**: Contains the database schemas.
  - **services**: Implements the core functionalities.
  - **middleware**: Includes authentication and error handling middleware.
  - **config**: Configuration files for database and environment variables.
  - **types**: TypeScript types and interfaces.

### Frontend

- **src**: Contains the main application code.
  - **pages**: Different pages of the application.
  - **components**: Reusable UI components.
  - **services**: Functions for API calls.
  - **styles**: CSS styles for the application.
  - **types**: TypeScript types and interfaces.

## Getting Started

To get started with OptiCV.ai, clone the repository and install the necessary dependencies for both the backend and frontend.

### Prerequisites

- Node.js
- PostgreSQL

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the backend directory and install dependencies:
   ```
   cd backend
   npm install
   ```

3. Navigate to the frontend directory and install dependencies:
   ```
   cd frontend
   npm install
   ```

4. Set up the database and environment variables as specified in the configuration files.

5. Start the backend server:
   ```
   npm start
   ```

6. Start the frontend application:
   ```
   npm start
   ```
